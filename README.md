# SolanaPage

SolanaPage is a small, modular new Tab page.

## Features

- Fully modular
- Visually appealing
- Easy to customize

## Requirements

- Node 6.6
- Playerctl (for media controls module)
- GraphicsMagick 1.3.25 (for bookmarks module)
- forever `npm install forever -g`
